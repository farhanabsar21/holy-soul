## Project

Holy Soul is a learning oriented platform for Holy Quran.

## Tech Used

- Next Js
- Typescript
- Tailwind
- Axios

## Getting Started

- clone the repo from github

```bash
git clone https://github.com/farhanabsar21/holy-soul.git
```

- install dependencies

```bash
npm i
# or
yarn install
```

- Then run the development server:

```bash
npm run dev
# or
yarn dev
```
